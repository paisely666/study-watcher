# Study Watcher（督学小工具）

一个面向大一学生的本地网页学习监督工具。帮你把"我打算学"变成"我真的学了"。

## 功能列表

- **首页**：今日日期、今日课程、今日任务、当前任务、安排合理性评价、睡觉按钮
- **课表管理**：手动录入课程（课程名、星期、时间、地点、学分），支持增删查
- **任务备忘录**：新增任务（内容、预测用时、预计执行时间），标记完成/推迟/摸鱼/放弃
- **安排合理性评价**：基于任务时长、时间段、课程学分、历史完成率自动评价
- **定时追问**：到达预计执行时间弹出提醒，支持正在学/准备开始/推迟/摸鱼/放弃
- **奖励与补救规则**：自定义奖励规则和补救规则，睡觉时自动检查触发
- **每日复盘**：按日期查看执行情况，红黄绿可视化表格，系统建议
- **睡觉总结**：手动或 24:00 自动展示当日完成情况、奖励、补救提醒

## 使用方法

1. 直接用浏览器打开 `index.html`
2. 不需要安装任何东西，不需要后端服务
3. 建议用 Chrome 或 Edge 打开

推荐使用流程：

1. 先在「课表」页添加本周课程
2. 在「任务」页添加今天的学习任务（必须填写预计执行时间）
3. 回到「首页」查看今日概览和安排评价
4. 任务到时间后页面会弹出追问弹窗
5. 学习结束后点击「睡觉」查看今日总结
6. 去「复盘」页查看历史执行情况

## 数据存储说明

所有数据存储在浏览器 localStorage 中，key 列表：

| Key | 说明 |
|-----|------|
| courses | 课程数据 |
| tasks | 任务数据 |
| logs | 执行日志 |
| rewardRules | 奖励规则 |
| rescueRules | 补救规则 |
| sleepSummaries | 每日总结 |

数据只在本地浏览器中，清除浏览器数据会丢失。建议定期导出备份。

## 第一版限制

- 不使用 React / Vue 等框架
- 不使用后端和数据库
- 不接入 AI API
- 不支持云同步
- 不支持浏览器通知（使用页面内弹窗）
- 任务相似度匹配使用简单关键词规则
- 安排评价使用纯规则判断
